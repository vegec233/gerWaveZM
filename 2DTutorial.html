Single Sine Wave

A sine wave, sinusoidal wave, or just sinusoid is a mathematical curve defined in terms of the sine trigonometric function, of which it is the graph. It is a type of continuous wave and also a smooth periodic function. It occurs often in mathematics, as well as in physics, engineering, signal processing and many other fields.

This type of wave is periodic and smooth. Hence, it is a good choice to simulate calm and regular liquid surface motion, for example, a well water surface ripple.

y = Amplitude * sin( x )

Sum of Sine Waves

By summing up multiple sine waves with different frequency, wave length or amplitude, a more random motion can be created. This is desirable for simulating a lake or stream surface. However, it is still not good enough for a sea surface.

Single Gerstner Wave

Similar to sine wave but points on gerstner wave is moving around circles instead of straight up and down oscillation. As a circle can be drawn through (cos(x),sin(x)), while x is increasing. This creats a steep crest and a smooth trough. Now this looks more like a sea wave.

However, when the wave amplitude is too high, the shape of the wave will become abnormal. Hence, be careful with the value of amplitude when using it.

X-Coordinate = Amplitude * cos ( (2 * Pi / waveLength) * (Offset) )

Y-Coordinate = Amplitude * sin ( (2 * Pi / waveLength) * (Offset) )

Sum of Gerstner Waves

Similar to sine wave examples above, by adding up different gerstner waves can create a random wave motion. Now it is an ideal wave motion for a sea surface simulation! However, the abnormal shape of the wave will stil occur when the amplitudes goes to high. Do be careful with these parameters. But luckily, when you need to increase the amplitude is too high, you can fix its shape by adjusting the frquency and period of the wave. You will get to try it in the 3D simulation scene in this site.
